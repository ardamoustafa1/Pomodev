# Pomodev Chrome Extension

🍅 Pomodev Pomodoro Timer Chrome eklentisi

## Kurulum

1. Chrome'da `chrome://extensions/` adresine gidin
2. Sağ üstten "Geliştirici modu"nu açın
3. "Paketlenmemiş uzantı yükle" butonuna tıklayın
4. Bu `chrome-extension` klasörünü seçin

## Özellikler

- ⏱️ Pomodoro / Kısa Mola / Uzun Mola modları
- 📊 Günlük ve haftalık istatistikler
- 🔥 Streak takibi
- 🎮 XP ve seviye sistemi
- 🔔 Masaüstü bildirimleri
- 💾 Veriler chrome.storage'da saklanır

## İkonlar

İkon dosyalarını oluşturmanız gerekiyor:
- `icons/icon16.png` (16x16)
- `icons/icon48.png` (48x48)
- `icons/icon128.png` (128x128)

Ana sitedeki favicon'u farklı boyutlarda kullanabilirsiniz.

## Geliştirme

```bash
# Değişikliklerden sonra eklentiyi güncelleyin:
```
